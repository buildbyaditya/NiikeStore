# Nike x Marvel: Futuristic Concept Store 🚀⚡

An immersive, futuristic Nike e-commerce concept store showcasing limited-edition sneaker drops inspired by Marvel's **Iron Man** and **Black Panther**. 

This entire project was built as a personal challenge to push the absolute limits of modern frontend styling: **100% pure HTML and CSS with absolutely zero JavaScript.**

## 🌟 Key Features

* **Iron Man Special Edition Hub:** A high-tech, Stark Industries-inspired user interface featuring mechanical hover effects, neon glow transitions, and arc-reactor thematic styling.
* **Black Panther Limited Edition Hub:** A sleek, Wakandan-inspired dark mode interface utilizing vibrant purple vibranium energy flows and sharp layout transitions.
* **No-JS Interactive UI:** Dropdowns, image sliders, tab switching, and modal effects are achieved purely through creative HTML structure and CSS state handling (such as the checkbox hack and target pseudo-classes).
* **Advanced CSS Animations:** Immersive `@keyframes` animations handling smooth transitions, scaling, glowing borders, and floating interactive elements.

## 🛠️ Technologies Used

* **HTML5:** Structured semantic markup optimized for styling hooks.
* **CSS3:** Custom properties (variables), Grid, Flexbox, `@keyframes`, transitions, and pseudo-states (`:hover`, `:checked`, `:target`, `:focus`).

## 🚀 Highlights of the Pure-CSS Architecture

Without relying on JavaScript event listeners, this project successfully implements:
1.  **State Management via CSS:** Utilizing hidden `<input type="checkbox">` and `<input type="radio">` tags paired with the sibling selector (`~`) to toggle product views and themes.
2.  **Fluid Motion Profiles:** Customized `cubic-bezier` timing curves to replicate premium, high-tech screen transitions.
3.  **Responsive Layouts:** Seamless adaptation across desktop and mobile screen sizes using fluid media queries.
